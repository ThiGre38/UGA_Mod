(Cours en Français)

# Installation

Choisissez un emplacement adéquat sur votre machine et tapez les commandes suivantes:

```
git clone https://github.com/ThiGre38/UGA_Mod.git
cd UGA_Mod
```

Ouvrez le fichier `index.html`
Naviguez sur le site

# Usage

Le contenu du dossier UGA_Mod peut être utilisé localement ou placé en totalité sur un serveur web pour un accès en ligne.

# Suppression

- Placez le dossier dans sa totalité à la corbeille en utilisant votre gestionnaire de fichiers
- Utilisez les commandes suivantes:

```
cd UGA_mod
rm -Rf *
```
