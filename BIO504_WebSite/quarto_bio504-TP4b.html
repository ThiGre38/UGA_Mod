1 Introduction à la modélisation structurale

1.1 Objectifs du TP

  1. Découvrir la base de données PDB
  2. Faire ses premiers pas avec AlphaFold (AlphaFold2)
  3. Utiliser AlphaFold server (AlphaFold3) pour déterminer une structure protéique
  4. Visualisez les structures et interagir avec elles

1.2 Découvrir la base de donnée PDB

  • Dans Google, faites une recherche avec les mots clés suivants protein, structure 3D, database

    • Quels sont les résultats obtenus ?
    • Que se passe-t-il quand vous combinez les mots clés entre eux ?
    • Recherchez les résultats pointant vers ce que vous pensez être des bases de données de structures de protéines. Quelles sont ces bases ?
    • Notez les sites web correspondant

  • Rendez-vous sur ces différents sites

    • Sont-ils équivalents ?

  • Rendez-vous sur le site des structures proposées par le NCBI (il doit faire partie des sites que vous venez d’identifier)

    • Notez les services proposés par ce site
    • Passez maintenant sur le site généraliste des structures protéiques (il doit également faire partie des résultats obtenus à l’étape précédente)
    • Quelle a été votre porte d’entrée sur le site ? (indiquez le lien hypertexte affiché dans la barre d’adresse du navigateur)
    • Les services proposés sont-ils différents ?

1.3 À la découverte d’une séquence protéique

MGHFTEEDKATITSLWGKVNVEDAGGETLGRLLVVYPWTQRFFDS
FGNLSSASAIMGNPKVKAHGKKVLTSLGDAIKHLDDLKGTFAQLS
ELHCDKLHVDPENFKLLGNVLVTVLAIHFGKEFTPEVQASWQKMV
TGVASALSSRYH
  • Identifiez cette séquence, quel programme utilisez-vous ?
  • Quels paramètres sont judicieux ?
  • De quel organisme provient la séquence ?
  • Quelles sont les structures existantes pour cette protéine dans la base de donnée RCSB
  • Sélectionner une structure et télécharger le fichier PDB correspondant dans votre dossier de travail
  • Ouvrez ce fichier dans un éditeur de texte et observez son organisation
  • Refermez-le SANS modification

1.4 Premiers pas avec AlphaFold

  • Recherchez un modèle de la protéine que vous venez d’identifier dans AlphaFold ?
  • Quel est son code d’accès ? Quelle est sa version ?
  • Affichez le modèle, quelles sont les informations listées sur la page ?
  • Analysez chaque rubrique et donnez-en la signification
  • Récupérez le modèle au format PDB et stockez-le dans votre dossier de travail

1.5 Prédiction de repliement

Afin de tester les capacités des logiciels de prédiction de structure, nous allons utiliser la dernière version du serveur AlphaFold (AlphaFold3).

L’accès au serveur nécessite une adresse mail gmail pour se connecter.

Une fois connecté au serveur d’AlphaFold3 avec votre adresse gmail, vous ouvrez la page suivante :


Vous pouvez copier-coller la séquence à modeler dans la boite centrale (la version gratuite vous autorise 30 calculs par jour). Une fois la séquence collée, elle est automatiquement mise en forme par l’interface


  • Quelles sont les options à votre disposition ?
  • Pour lancer le calcul, que devez-vous faire ?
  • Les temps de calculs peuvent aller de 2 à 10 minutes
  • Quel est le résultat obtenu ?
  • Quels sont les caractéristiques du modèle affiché ? Expliquez les différentes mesures.
  • Téléchargez le résultat du calcul d’AF3 :
    • Combien de modèle vous sont proposés ?
    • Analysez les noms des fichiers et déduisez leur contenu
    • Quel modèle vous semble le meilleur ? Vous allez l’utiliser pour l’observation de la structure

1.6 Observation de la structure

Les modèles sont présentés sous un format de fichier spécifique pour les structures. Différents logiciels disponibles sur le net peuvent afficher ces fichiers, parmi ceux-ci, vous trouverez des logiciels codés en python à compiler vous-même, mais il existe également des logiciels prêts à l’emploi dont PyMol (codé en python également, payant mais proposant une licence gratuite pour le monde académique)


et ChimeraX (gratuit).

Pour des raisons de commodité d’installation et de gestion des licences, nous utiliserons ChimeraX pour visualiser les modèles. Vous pouvez le télécharger à l’adresse suivante ChimeraX - Download, puis l’installer.

Choisissez la version correspondant à votre système et procédez à l’installation en suivant les indications (acceptez la licence d’utilisation pour démarrer le téléchargement).

Après installation, l’ouverture du logiciel vous propose une interface semblable à celle-ci :


Les commandes sont accessibles par des menus ou des boutons


mais peuvent aussi être programmées ainsi que de nombreux paramètres dans des scripts. À cet effet, vous avez une ligne de commande sous le logiciel.


La partie droite de la fenêtre vous permet d’obtenir des informations sur le modèle affiché, de sélectionner des parties de structure, et d’accéder à l’aide en ligne.

  • Trouvez le moyen d’ouvrir le modèle proposé par AF3 que vous avez sélectionné.
  • Combien de chaînes polypeptidiques contient ce modèle ?
  • Quelle est sa séquence ?
  • Identifiez les extremités NH2 et COOH
  • Visualisez les chaines, les atomes et faites varier les styles
  • Affichez la structure en mode rainbow, combien de régions différentes identifiez-vous ?
  • Visualisez les régions hydrophobes et les répartitions des charges électrostatiques

Vous allez maintenant vérifier la validité de votre calcul en comparant le modèle AF3 avec le modèle AF2, téléchargé au point 1.4

  • Pour cela, ouvrez les deux modèles dans ChimeraX.
  • Les deux modèles sont-ils superposés ?
  • Que pensez-vous de la validité du calcul dans AF3 ?
  • Que devez-vous faire ?

Pour terminer, comparez maintenant votre modèle calculé par AF3 au modèle RCSB obtenu par expérience, sont-ils superposables ?

1.7 Pour aller plus loin (Travail personnel)

  1. Récupérez les séquences des alphas et des betas et faites calculer à AlphaFold server l’interaction possible entre :
    • Une protéine de la famille \(\alpha\) et une protéine de la famille \(\beta\)
    • Deux protéines de la famille \(\alpha\) et deux de la famille \(\beta\)
  2. Recherchez une structure quaternaire de l’hémoglobine fonctionnelle (téléchargez les fichiers de structure adéquats)
    • Utilisez ChimeraX pour comparer les résultats des calculs avec le modèle de l’hémoglobine fonctionnelle
    • Justifiez vos déductions et trouvez une explication simple
  3. Existe-t-il une différence de structure entre l’hémoglobine foetale sauvage (complexée avec l’oxygène) et l’hémoglobine foetale complexée avec le monoxyde de carbone ?
